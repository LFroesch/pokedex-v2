## Quick Start

1. **Install dependencies**
   ```bash
   # Frontend
   cd frontend
   npm install
   ```

2. **Run the backend**
   ```bash
   # Backend (from root)
   cd backend
   go run . -web -port 8080
   ```

3. **Run the frontend**
   ```bash
   # Frontend (new terminal)
   cd frontend
   npm run dev
   ```
